NeonBans

✨ Features 📥 Installation 📜 Commands ⚙️ Config 🔑 Permissions 📬 Webhook

✨ Features

  • Ban, Kick, Mute, Warn, Tempban, Tempmute
  • Webhook support for Discord
  • Local SQLite storage
  • Configurable punishment messages
  • Auto-warning punishment thresholds

📥 Installation

  1. Download the plugin from GitHub
  2. Place NeonBans.jar into your /plugins folder
  3. Restart your Minecraft server
  4. Edit the config.yml to your preferences

📜 Commands

  • /ban <player> <reason>
  • /kick <player> <reason>
  • /mute <player> <duration>
  • /warn <player> <reason>
  • /tempban <player> <duration> <reason>
  • /tempmute <player> <duration>
  • /unban <player>
  • /unmute <player>

⚙️ Example Config

# NeonBans configuration

broadcast-ban-messages: true
ban-message: "&cYou are banned from this server! Reason: %reason%"

webhook:
  enabled: true
  url: "https://discord.com/api/webhooks/your_id/your_token"

messages:
  ban: |
    🚫 **Ban Issued**
    **Player:** %player%
    **Reason:** %reason%
