NeonBans

Advanced Ban, Mute, Kick, and Warning System for Minecraft Servers

GitHub Direct Download

🔧 Features

  • Ban, kick, mute, warn system
  • Webhook support for Discord
  • SQLite local storage, no setup needed
  • Custom messages for punishments
  • Configurable thresholds for warnings
  • Simple, clean config.yml

📥 Installation

  1. Download the plugin
  2. Place it in your /plugins folder
  3. Restart the server
  4. Edit the config to your liking

📜 Commands

  • /ban <player> <reason> – Ban a player
  • /kick <player> <reason> – Kick a player
  • /mute <player> <duration> – Mute a player
  • /warn <player> <reason> – Warn a player
  • /unban <player> – Unban a player

⚙️ Example Config

# NeonBans configuration

broadcast-ban-messages: true
ban-message: "&cYou are banned from this server! Reason: %reason%"

webhook:
  enabled: true
  url: "https://discord.com/api/webhooks/your_id/your_token"

messages:
  ban: |
    🚫 **Ban Issued**
    **Player:** %player%
    **Reason:** %reason%

🔑 Permissions


neonbans.ban - Allows banning
neonbans.kick - Allows kicking
neonbans.mute - Allows muting
neonbans.warn - Allows warning
neonbans.admin - Full access
